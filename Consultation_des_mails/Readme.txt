Consulation de messages

Ce projet consiste à visualiser les mails présents dans la base Xedix.

Prérequis:
-Avoir importé le fichier XML présent dans le dossier "xedix" dans la base.
-La balise <id> du fichier XML doit être un identifiant unique d'un mail.
-Les fichiers présents dans le dossier "interface_mail" peuvent être déployés.
_Lancer dans un navigateur web "main.php".
